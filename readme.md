## Golavel Cache

Golavel Cache allows you to implement a data-store agnostic caching system 
via a common interface. It was inspired by the Laravel Cache System. 
It is currently a work in progress.

## Supported Stores

- Redis
- Memcache

## Future Stores 

- Gocache 
- BlockDB
